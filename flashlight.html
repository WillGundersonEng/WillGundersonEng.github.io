SolidWorks Flashlight Assembly

Project Type: CAD / Design for Manufacturing

Created a full 3D flashlight assembly in SolidWorks as a final project in Engineering Graphics and Design. Each component (body, lens, cap, reflector, button, and battery contact) was modeled individually, dimensioned, and assembled using proper mating constraints. Rendered final assembly views and exploded diagrams for presentation.

Key Highlights:

  • Designed all 8 parts to specified tolerances in SolidWorks.
  • Applied appropriate materials and surface finishes for each part.
  • Created technical drawings and dimensioned blueprints for each component.
  • Produced high-quality renders using SolidWorks Visualize.

Software Used: SolidWorks 2024 (Student Edition)
